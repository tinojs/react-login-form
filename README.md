## Features
- Email format validation
- Password length validation (>= 6)
- Friendly error/success messages
- Demo credentials button (for quick testing)
- Responsive and modern UI

## Tech
- React (Create React App)
- CSS (simple, responsive styles)

## Run locally

```bash
# clone
git clone https://github.com/tinojs/react-login-form.git
cd react-login-form/client

# install & start
npm install
npm start
```
